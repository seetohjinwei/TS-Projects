Tasks

by Jin Wei

    • Tip: Enter works for both adding and removing
    • Type your task in the input field then press +
    • Click on the task to mark as done/undone
    • Click on waste bin to remove task
    • To bulk remove tasks, examples shown below, then, press DEL
      • "2": clears index 2
      • "2-": clears index 2 to end of list
      • "-2": clears from start of list to index 2
      • "2-4" clears from index 2 to 4
      • ignores any other entry
    • Auto-saves when you refresh or close tab
    • Save to manually save
    • Recover to go back to browser's last save
    • Import/export from a hash in the input field
